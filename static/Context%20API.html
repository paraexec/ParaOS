Context API

19th February 2022 at 12:20pm
API

The purpose of this API is to provide arbitrary context information to other APIs. For example, if one wants to use other APIs in a particular transaction context, they can attach that transaction to the context and use the context to make a call.

interface Context {
    set<T>(self, t: T);
    get<T>(self) Maybe<T>;
}

interface ContextProvider {
   newContext(self) Context;
}
