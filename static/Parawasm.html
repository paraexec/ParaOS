Parawasm

21st February 2022 at 7:32pm

WebAssembly to native compiler.

Currently supports x86_64 only and is incomplete. It is based on an understanding of assumptions that go into ParaOS' design. For example, it is currently expected that since we control all code generation, the code compiled from WebAssembly modules will be able to run in privileged mode, allowing system calls to be regular procedure calls.
